== Description ==

Openmeetings Wordpress integration widget.
OpenMeetings is a free browser-based software that allows you to set up instantly a conference in the Web. You can use your microphone or webcam, share documents on a white board, share your screen or record meetings. It is available as hosted service or you download and install a package on your server with no limitations in usage or users.

Please visit us for more information and updates at:

http://www.eyecatchers.co.il/openmeetings-widget-wordpress-plugin/
Openmeetings hosting at: http://www.linux-hosting.co.il/

Openmeetings has now been released more advanced plugin for Wordpress for a few dollars. I suggest you go grab a copy at:
http://code.google.com/p/openmeetings/wiki/WordpressIntegration


== Installation ==

1. Upload `openmeetings` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the widget in your templates.

PHP SOAP extention must be enabled!

You must have an Openmeetings Server up and running inorder to produce hash keys.
For installation go to the Openmeetings webpage:
http://code.google.com/p/openmeetings/
If you need installation and integration please contact me.

== Frequently Asked Questions ==

Q: I don't see the "Chat now!" text, only the header!
A: Make sure you have the SOAP extention for PHP enabled. http://php.net/manual/en/book.soap.php

Q: Not working!
A: Make sure that server is up and running and ports are opened.
